# 🐳 SQL & Docker Practice for QA

Учебный проект для отработки навыков работы с базами данных (PostgreSQL) в изолированном окружении Docker. Предназначен для практики написания запросов, которые встречаются в реальных задачах тестировщика (Яндекс Лавка/Маркет).

---

## 🛠 Технологический стек
* **Database:** PostgreSQL 15
* **Infrastructure:** Docker, Docker Compose
* **Scenarios:** Users & Orders (Relational Data)

---

## 🚀 Как запустить окружение
1. Склонируйте репозиторий.
2. Убедитесь, что Docker Desktop запущен.
3. Выполните команду в терминале:
   ```bash
   docker compose up -d
   ```
   *База будет автоматически создана и наполнена тестовыми данными из файла `init.sql`.*

---

## 🔍 Структура данных
Проект имитирует базу данных интернет-магазина:
- **`users`**: Таблица пользователей (id, name, email, registration_date).
- **`orders`**: История заказов (order_id, product_name, price) со связью по `user_id`.

### Реализованные сценарии (SQL):
В ходе практики были отработаны:
- **Фильтрация:** Поиск пользователей по маске (`LIKE 'N%'`).
- **Агрегация:** Подсчет общей суммы заказов пользователя (`SUM`, `GROUP BY`).
- **Объединение таблиц:** Склейка пользователей с их заказами (`INNER JOIN`).
- **Поиск пропусков:** Нахождение «пустых» пользователей без заказов через `LEFT JOIN ... WHERE IS NULL`.
